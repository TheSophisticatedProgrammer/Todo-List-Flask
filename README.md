# Todo-List-Flask

A simple todo list web app built with Python and Flask.

## Features

- Add tasks
- Mark tasks as complete with a checkbox
- Delete tasks
- Clean responsive UI with CSS

## Tech Stack

- Python 3
- Flask
- Jinja2 templates
- HTML/CSS

## How to Run

1. Install dependencies:
```
pip install flask
```

2. Run the app:
```
python app.py
```

3. Open your browser and go to `localhost:5000`
